# WireGuard VPN内网配置

**提示**：所有`*.ana`及其子域名均为内网域名，需要连接内网VPN才能访问。

## 配置文件

如需使用内网VPN，请向部长申请账号。

客户端配置模板：`NAME - ana.conf`

下载配置模板后，需要修改以下内容：

 - 修改文件名中的`NAME`和`<Your Name>`为你的姓名全拼便于识别。

 - 修改`<Your Private Key String>`为你的私钥字符串。

 - 修改`<Your Interface IP>`为部长分配给你的IP。

最后，部长将你的公钥和IP记录到本项目中。

## 客户端

目前推荐使用`TunSafe`的客户端。
